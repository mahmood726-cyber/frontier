## E156 Abstract (CURRENT BODY)

Can high-dimensional global health datasets be synthesized with distribution-free coverage guarantees while preserving population alignment? We implemented a 2026 frontier pipeline using conformalized meta-learning and Wasserstein-over-Wasserstein distance to synthesize IHME burden data anchored by ClinicalTrials.gov randomized trials. The framework provided 90 percent finite-sample coverage guarantees for regional statin effects by reweighting noisy observational labels against clean trial anchors. The global Bayesian posterior for statin efficacy was 0.808 (90% highest-density interval 0.780 to 0.836), while geometric alignment identified China as the population most similar to the RCT anchor cohorts. All estimates are certified via the TruthCert protocol, linking every conformalized prediction set to source evidence hashes. This methodology shows that distribution-free uncertainty quantification is feasible in heterogeneous global settings without locking the analysis to a narrow parametric model. The current implementation remains limited by synthetic benchmarking and by the availability of comparable anchor trials.
