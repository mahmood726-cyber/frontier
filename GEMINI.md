## Purpose
Ship a state-of-the-art **2026 Frontier Statistical Synthesis** of Cardiovascular Disease (CVD) as an **E156 micro-paper + GitHub repo + interactive HTML dashboard**.

## Session Workflow
Before you use any tool or make changes, briefly say what you're about to do, then do it. After each tool call, summarize what you found and what's next.

## Statistical Framework: 2026 Frontier Methods
- **Conformalized Meta-Learning (CML):** Reweighting noisy IHME data against CT.gov anchor trials for distribution-free coverage.
- **Wasserstein-over-Wasserstein (WoW) Distance:** Geometric population alignment via Optimal Transport.
- **Stein Variational Gradient Descent (SVGD):** Particle-based Bayesian inference to capture complex posteriors.

## Non-negotiables

4. **Fail-closed**: if validation incomplete, REJECT + reasons.
5. **Determinism**: fixed seeds (seed=42), stable sorting.
